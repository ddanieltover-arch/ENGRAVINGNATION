# KPI Dashboard Definitions

## Reporting Cadence

- Weekly: traffic, conversions, Core Web Vitals trend
- Monthly: keyword and ranking movement, authority and links
- Quarterly: GEO/AI citation share and content ROI

## Dashboard Views to Build

1. **Executive Summary:** sessions, revenue/leads, conversion rate, top movers.
2. **SEO Performance:** impressions, clicks, CTR, average position by page and query.
3. **Technical Health:** CWV metrics, crawl/index alerts, 4xx/5xx trend.
4. **Content ROI:** traffic and conversions by article/product category.
5. **GEO Visibility:** monthly AI citation checks by priority query set.

## Alert Thresholds

- Ranking drop > 5 positions on any priority keyword.
- LCP > 2.5s on any primary landing page for 7-day average.
- Organic sessions down > 15% week-over-week.
- New indexed utility/transaction pages detected.
